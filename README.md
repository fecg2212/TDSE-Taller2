# TDSE - Taller 02
## Heart Disease Risk Prediction with Logistic Regression

Implementación desde cero de un modelo de regresión logística para predecir el riesgo de enfermedad cardíaca a partir de variables clínicas, utilizando Python, análisis exploratorio de datos, visualización de límites de decisión, regularización y una exploración de despliegue en Amazon SageMaker.

## Getting Started

Estas instrucciones permiten obtener una copia del proyecto y ejecutarlo localmente para fines académicos, experimentación y análisis del modelo. El desarrollo se realiza principalmente en un Jupyter Notebook.

## Prerequisites

Para ejecutar este proyecto se requiere:

- Python 3.8 o superior  
- Git  
- Jupyter Notebook o Jupyter Lab  

### Librerías necesarias

- NumPy  
- Pandas  
- Matplotlib
